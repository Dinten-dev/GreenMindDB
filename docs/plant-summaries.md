## Confidence Score Methodology

| Score | Meaning |
|-------|---------|
| 0.95 | Multiple university sources agree exactly |
| 0.90 | Sources agree with minor variation |
| 0.85 | Sources differ slightly; conservative range used |
| 0.80 | Limited sources; based on best available data |
| <0.80 | Single source or derived value |

All values without reliable sources were left empty rather than estimated.
